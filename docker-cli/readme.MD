# 기본명령
    - 클라이언트/서버 측이 채택한 아킥텍쳐를 보여줌
        - docker version
            - 클라이언트(데탑용), 서버(도커엔진, 내부에존재, 리눅스)
    - 도커에 현재 현황 정보 제공 (실행환경, 컨테이너수, 도커 버전, 스토리,..)
        - docker system info
        - 디스크 이용 상황
            - docker system df
